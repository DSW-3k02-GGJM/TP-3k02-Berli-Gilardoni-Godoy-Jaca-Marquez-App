# add

## Descripción
Este método permite la creación de una marca.

---
## Solicitud
### Parámetros para la solicitud
* brandName: string, **obligatorio**. Nombre de la marca.

---
## Respuesta
### Parámetros de respuesta
* message: string. Mensaje de confirmación.
