# TP-3k02-Berli-Gilardoni-Godoy-Jaca-Marquez-BE-App

## Compilación y ejecución de Typescript (npm y sin configuración)
### Compilación
1. Desde la carpeta 'backend' abrir una consola.

2. Escribir y ejecutar el comando: 

```bash
   npm run build
```
### Ejecución
1. Desde la carpeta 'backend' abrir una consola.

2. Escribir y ejecutar el comando: 

```bash
   npm run start:dev
```
